# Project Constitution

## Project Map & State Tracking
- **Current Phase:** Phase 3 (Architect / Tools)

## Behavioral Rules
1. Prioritize reliability over speed.
2. Never guess at business logic.
3. Use the B.L.A.S.T protocol and A.N.T architecture.
4. If logic changes, update SOP before updating code.

## Architectural Invariants
- 3-Layer Architecture (SOPs, Navigation, Tools).
- Deterministic Python scripts in `tools/`.
- React/Vite + Vanilla CSS for the web interface.
- No global side-effects outside of defined payload delivery.
