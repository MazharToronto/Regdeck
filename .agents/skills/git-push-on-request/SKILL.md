# Git Push Control Skill

## Instructions
- **Do NOT push changes automatically to Git.**
- Perform local edits, verification, and local commits as needed, but **NEVER** run `git push` unless the user explicitly instructs you to push changes.
- Wait for explicit user confirmation/request (e.g., "push the changes into git") before executing `git push`.
