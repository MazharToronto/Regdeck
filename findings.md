# Findings

## Research, Discoveries, and Constraints
- **North Star:** Responsive web app to create database entries, view records, and generate reports.
- **Integrations:** Supabase (Database, API, Authentication, Role-based Access Control).
- **Source of Truth:** Supabase PostgreSQL Database.
- **Delivery Payload:** Supabase Database (Write), API responses for UI rendering/reporting.
- **Behavioral Rules:** 
  - Left-side navigation panel with icons.
  - Supabase Authentication required.
  - Role-based screen access.
  - Fully responsive design for all devices.
- **Contextual Discovery:** The workspace is named `InvoiceGen`, implying the predefined fields likely revolve around Invoice tracking (e.g., Client, Amount, Status).
